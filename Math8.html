Problem 1. In this exercise, we will try to understand the Second Derivative Test by proving the first case. Let f(x, y) be a differentiable function at the point (a, b) such that fx(a, b) = fy(a, b) = 0 and all second partial derivatives of f are continuous at (a, b).

(1) Show that Duf = 0 for all unit vectors u = ⟨u1, u2⟩, and that

D2uf = Du(Duf) = fxxu12 + 2fxyu1u2 + fyyu22

(2) Recall D = fxxfyy - f2xy. By completing the square, show that D2uf(a, b) > 0 if fxx(a, b) > 0 and D(a, b) > 0. This shows Duf is increasing in all directions u. Combining Duf(a, b) = 0, we see that f is increasing at (a, b) in all directions, so f(a, b) is a local minimum.
